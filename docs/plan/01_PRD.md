# PRD — 온천관광호텔 경영 대시보드

> **버전:** v0.1 | **Phase:** 1차 (과제 제출·발표용 프로토타입)

---

## 1. 서비스 목적

### 1.1 해결하려는 문제

| 문제 | 설명 |
|------|------|
| 데이터 분산 | 일별 매출·객실 청소·VOC가 Sheets·수기·여러 채널에 흩어져 보고 준비에 시간 소요 |
| 시야 분리 부족 | 일 운영(7일) / 월 결산 / 연간·계절 전략을 한 흐름으로 보기 어려움 |
| 이상 징후 대응 지연 | 매출 급락, 청소 미실시, VOC-시설 불만의 교차 신호를 수동 연결 |

### 1.2 주요 사용자

| 사용자 | 역할 | 1차 사용 목적 |
|--------|------|----------------|
| 경영진·호텔 대표 | 의사결정 | 월 KPI·연간 브리핑·전략 권고 |
| 운영팀장 | 일상 운영 | 최근 7일 실적, 객실 청소·검수 현황 |
| 마케팅·영업 | 수익 개선 | 사업부문(객실/온천/F&B) 구성·계절별 패턴 |
| 과제 심사·교수 | 평가 | 데이터 연동 흐름·대시보드 데모·외부 URL |

### 1.3 핵심 사용 시나리오

1. **아침 운영 브리핑** — 요약 탭 KPI → 실적 탭 최근 7일 추이
2. **월말 경영 보고** — 실적 탭 월별 실적 → 일자별 매출 → AX·인사이트 Executive Briefing
3. **품질 이슈 대응** — 객실운영 불량률 → VOC·AX 인사이트 → 운영 전략 카드
4. **과제 제출·발표** — 외부 URL → Sheets 원천 → 대시보드 탭 순서 데모

## 2. 제품 범위

### 2.1 Phase 1 (In Scope)

- Streamlit 웹 대시보드 5탭 (요약·실적·객실운영·VOC·AX·인사이트)
- Google Sheets 실데이터 연동 (매출·객실)
- VOC 샘플 데이터 (`data/voc.py`)
- 규칙 기반 AX 인사이트 + Executive 분석
- Streamlit Community Cloud 외부 공유 URL

### 2.2 Phase 1 (Out of Scope)

- 업무자동화 탭 UI (설계만 — [06_업무자동화_설계서.md](06_업무자동화_설계서.md))
- DB 연동·ETL (Phase 2)
- VOC 실연동·OTA API
- 서버 PDF 자동 생성
- 로그인·권한 관리

### 2.3 Phase 2 (로드맵)

- Next.js + Vercel 프론트
- Supabase Postgres + ETL
- 업무자동화 MVP
- 모바일 PWA·알림

## 3. 출력 결과 정의

### 3.1 요약 (`?tab=summary`)

| 출력 | 입력 | 형식 |
|------|------|------|
| KPI 4종 | revenue, rooms | 카드 + MoM/WoW % |
| 경고 배지 | revenue | 카테고리 -10% 이하 |
| Daily Performance | revenue 최신월 | 2×2 KPI + 도넛 |

### 3.2 실적 (`?tab=performance`)

| 출력 | 형식 |
|------|------|
| 최근 7일 추이·구성·상세표 | 라인 + 누적막대 + 테이블 |
| 월별 매출 추이 | 라인 + 월 선택 |
| 전월 대비 | grouped bar + 표 |
| 일자별 매출 (선택 월) | 라인 + 일별 테이블 |

### 3.3 객실운영 (`?tab=rooms`)

불량률·미실시 건수, 담당자별 차트, 객실 리스트(필터)

### 3.4 VOC (`?tab=voc`)

평균 평점, 리뷰 수, 카테고리별 불만 차트

### 3.5 AX·인사이트 (`?tab=insights`)

| 섹션 | 출력 |
|------|------|
| Executive Briefing | 헤드라인 + bullet pills |
| 연중 매출 분석 | 계절·부문·분기 (역 12개월) |
| 마케팅·운영 전략 | 우선순위 카드 |
| AX 인사이트 | severity 카드 + 딥링크 |
| 리뷰 분석 | 평점·키워드·규칙/AI 요약 |

### 3.6 PDF / 월간보고서

| 항목 | 1차 | 2차 |
|------|-----|-----|
| 브라우저 인쇄 | 수동 가능 | print CSS |
| 서버 PDF | 미구현 | weasyprint / Next API |
| 월간보고서 자동화 | 설계만 | report_snapshots |
