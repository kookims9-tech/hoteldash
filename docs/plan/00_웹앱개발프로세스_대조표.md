## 3. 통합 PDF 스크린샷 vs 현행 구현

통합 PDF(2026-07-07 캡처)에 나타난 화면과 현행 코드 대조:

| PDF 화면 | 현행 탭 | 차이·비고 |
|----------|---------|-----------|
| KPI 4종 (월매출·점유율·ADR·주간매출) | `summary` | 일치 |
| Daily Performance (2×2 KPI + 도넛) | `summary` | PDF에는 미포함 → **현행이 확장** |
| 실적관리 최근 7일 | `performance` | 일치 |
| 주간 실적 (12주) | — | **제거됨** → 월별 실적 + 일자별 매출로 대체 |
| 월별 실적·전월 대비·구성비 | `performance` | 일치 (섹션명 "월별 요약" → "월별 실적") |
| 일자별 매출 (선택 월) | `performance` | **현행 추가** (PDF 후속 개선) |
| 객실 청소·검수 | `rooms` | UI 라벨 "상태" → "청소상태" |
| VOC 요약 | `voc` | 일치 (샘플 데이터) |
| AX·인사이트 (기본) | `insights` | **현행 확장**: Executive Briefing, 연중 분석, 마케팅/운영 전략 |
| 업무자동화 | — | 미구현 (설계서만) |
