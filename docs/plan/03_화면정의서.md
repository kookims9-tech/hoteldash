## 3. 요약 탭 (`summary`)

### 3.1 KPI 카드 (4종)

| KPI | 계산 | 비교 |
|-----|------|------|
| 월매출 | 당월 `total_revenue` 합 | 전월 대비 % |
| 점유율 | 당월 `occupancy_room` 평균 | 전월 대비 % |
| ADR | 객실매출 ÷ 판매객실 | 전월 대비 % |
| 주간매출 | 기준 주(월~일) 합계 | 전주 대비 % |

- 경고 배지: 카테고리별 전월 -10% 이하 (`kpi_category_alerts`)
- 함수: `render_kpis`, `compute_kpis`

### 3.2 Daily Performance

| 입력 | 출력 |
|------|------|
| `revenue` (최신 월) | 2×2 일별 KPI 카드 + 도넛 차트 |

- 함수: `render_daily_performance_panel`, `daily_performance`

---

## 4. 실적 탭 (`performance`)

> **참고:** 주간 실적(12주) 섹션은 제거됨. 월별 실적 + 일자별 매출로 대체.

### 4.1 실적관리 (최근 7일)

| 입력 | 출력 |
|------|------|
| `revenue`, `RECENT_DAYS=7` | 일별 추이 라인, 구성 누적막대, 상세 표 |

- 함수: `render_operations_panel`

### 4.2 월별 실적

| 입력 | 출력 |
|------|------|
| `revenue`, `selected_month` (session) | 월별 추이, 전월 대비, 구성비 |

- 함수: `render_monthly_panel`, `render_mom_panel`

### 4.3 일자별 매출

| 입력 | 출력 |
|------|------|
| `revenue`, `selected_month` | 선택 월 1일~말일 라인 + 테이블 |

- 함수: `render_monthly_daily_panel`

### 4.4 쿼리 파라미터

- `?tab=performance&category=room|spa|fb` — 부문 포커스 스크롤

## 5. 객실운영 탭 (`rooms`)

| 입력 | 출력 |
|------|------|
| `rooms` | 불량률, 미실시 건수, 담당자별 차트 |
| 필터: 담당자, 청소상태, 검수결과 | 객실 리스트 테이블 |

- 함수: `render_room_panel(show_room_list=True)`
- UI 라벨: `status` → **청소상태**

---

## 6. VOC 탭 (`voc`)

| 입력 | 출력 |
|------|------|
| `voc_summary()` | 평균 평점, 리뷰 수, 불만 카테고리 차트 |

- 캡션: AX·인사이트 탭 링크
- 함수: `render_voc_panel`

---

## 7. AX·인사이트 탭 (`insights`)

| 섹션 | 함수 | 출력 |
|------|------|------|
| Executive Briefing | `render_executive_briefing_section` | 헤드라인·bullets |
| 연중 매출 분석 | `render_annual_revenue_report` | 계절·부문·분기 |
| 마케팅 전략 | `render_marketing_strategy_section` | 전략 카드 |
| 운영 전략 | `render_operations_strategy_section` | 전략 카드 |
| AX 인사이트 | `render_ax_insights_section` | severity 카드 |
| 리뷰 분석 | `render_review_analysis_section` | 키워드·AI 요약 |

- 분석 기간: 역 12개월 (`lib/executive_analysis._analysis_period`)
