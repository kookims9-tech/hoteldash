# Google Sheets 연동 설명

## 연동 개요

대시보드는 Google Sheets API 키 없이 **공개 CSV Export URL**로 데이터를 읽습니다.

```
시트(웹) → CSV Export → Python (lib/sheets.py) → pandas → Streamlit 화면
```

구현 파일: `lib/sheets.py`, 시트 ID 설정: `config.py` / `.env`

## CSV Export URL 형식

```
https://docs.google.com/spreadsheets/d/{시트ID}/gviz/tq?tqx=out:csv&sheet={탭명}
```

예시 (실적):

```
https://docs.google.com/spreadsheets/d/13SeEWtUh82azC8DYBtvY669KOoiVoYvCCWn9G_tSZDQ/gviz/tq?tqx=out:csv&sheet=revenue
```

## 권한·설정 체크리스트

발표 전 아래를 확인하세요.

- [ ] 시트 공유: **「링크가 있는 모든 사용자」보기** 이상
- [ ] 탭 이름 일치: `revenue`, `Rooms`
- [ ] 컬럼명이 코드와 동일 (영문, 첫 행 헤더)
- [ ] 매출 값 단위: **천원** (대시보드 표기와 일치)
- [ ] 발표 PC에서 시트 URL 접속·대시보드 데이터 로드 테스트

## 발표 시 설명 포인트

1. **원천 데이터** — 호텔 실무에서 쓰는 Google Sheets 그대로 사용
2. **자동 반영** — 시트 수정 후 브라우저 새로고침 시 대시보드 갱신 (캐시 TTL 60초)
3. **역할 분리** — 실적·객실은 시트, VOC는 프로토타입용 코드 데이터

## 데이터 로드 정책

- **실데이터 only:** Sheets 로드 실패 시 mock 데이터로 대체하지 않습니다.
- 화면에 오류 메시지를 표시하고 KPI·차트 계산을 중단합니다.
- 발표·배포 전 시트 공개 범위와 네트워크 접속을 반드시 확인하세요.

## 오프라인 백업 (권장)

네트워크 불안 시 대비: 시트 → **파일 → 다운로드 → CSV** 저장 후 `assets/`에 보관 (설명·자료용).
