# 발표 개요

## 과제물 제목

**온천관광호텔 실적관리·경영 대시보드 (Python Streamlit)**

## 한 줄 소개

Google Sheets의 일별 실적·객실 데이터를 자동 연동하여, **요약 KPI → 실적관리(최근 7일) → 월별·일자별 실적 → AX·인사이트**를 한 화면에서 확인하는 경영 대시보드입니다.

## 개발 목표

1. **실적관리 중심** — 최근 7일 일별 매출 추이·구성으로 단기 운영 현황 즉시 파악
2. **월별 경영 보고** — 월별 KPI, 전월 대비, 선택 월 일자별 드릴다운
3. **경영 분석** — 연중(역 12개월) 계절·부문·분기 분석, Executive Briefing
4. **운영 연계** — 객실 청소·검수 현황, VOC(고객의소리) 모니터링, AX 교차 인사이트

## 개발 취지

- 실적 데이터가 Google Sheets에 분산되어 있어 보고·분석에 시간이 소요됨
- 경영진 발표용 프로토타입으로 **데이터 원천(시트) → 가공(Python) → 시각화(Streamlit)** 흐름을 구현
- Cursor AI(Coding + Agent)를 활용해 요구사항 반영·UI 개선·오류 수정을 반복 수행

2. **실적** 탭 — 최근 7일 추이 + 월별 실적 + 일자별 매출

4. **VOC** 탭 — 평점·불만 카테고리
