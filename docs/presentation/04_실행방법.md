# 실행 방법

## 사전 요구사항

- macOS (또는 Python 3.10+ 지원 OS)
- Python 3.14 (프로젝트 `VENV` 기준)
- Google Sheets 보기 권한 (실적·객실 시트)

---

## 최초 1회 설정

```bash
cd /Users/admin/Documents/hoteldash
python3 -m venv VENV
source VENV/bin/activate
pip install -r requirements.txt
cp .env.example .env   # 선택: 시트 ID·호텔명 커스터마이즈
```

---

## 매번 실행 (발표 당일 포함)

```bash
cd /Users/admin/Documents/hoteldash
source VENV/bin/activate
streamlit run app.py
```

터미널에 표시되는 **Local URL**을 브라우저에서 엽니다.

```
Local URL: http://localhost:8501
```

포트는 `8501`, `8502` 등 환경에 따라 다를 수 있습니다. **터미널에 나온 주소를 사용**하세요.

---

## 외부 공유 (Streamlit Cloud)

과제 제출용 고정 URL이 필요하면 Streamlit Community Cloud를 사용합니다.

1. GitHub에 코드 push
2. https://share.streamlit.io → Repository 선택, Main file: `app.py`
3. Secrets에 `.streamlit/secrets.toml.example` 내용 입력
4. Deploy 후 `https://<app>.streamlit.app` URL 제출

**상세 체크리스트:** [docs/plan/04_배포_보안_가이드.md](../plan/04_배포_보안_가이드.md)

## 중요 사항

| 항목 | 설명 |
|------|------|
| 터미널 유지 | `streamlit run` 실행 중 터미널을 닫으면 접속 불가 (로컬) |
| localhost | 내 PC에서만 동작; Cloud 배포 시 외부 접속 가능 |
| 재실행 | Mac 재부팅·종료 후에는 위 명령을 다시 실행 |
| 데이터 갱신 | 브라우저 **새로고침(F5)** (캐시 TTL 60초) |
| 실데이터 only | Sheets 로드 실패 시 mock 대체 없음 — 오류 표시 후 중단 |

## 자주 발생하는 오류

### `ERR_CONNECTION_REFUSED` / Safari가 서버에 연결할 수 없음

- **원인:** Streamlit 서버가 꺼짐
- **해결:** 터미널에서 `streamlit run app.py` 다시 실행

### `ImportError: cannot import name 'altair'` 등

- **원인:** 의존성 미설치
- **해결:** `pip install -r requirements.txt`

### `데이터를 불러오지 못했습니다`

- **원인:** 시트 권한·네트워크·탭 이름 오류
- **해결:** `02_구글시트_연동설명.md` 체크리스트 확인

### Proxy 403 (로컬 IDE 환경)

- **원인:** `HTTP_PROXY` 환경변수
- **해결:** `lib/sheets.py`가 `trust_env=False`로 처리 (코드 반영됨)
