# 기술 스택 · 아키텍처

## 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 언어 | Python 3.14 | 백엔드·데이터 처리 |
| UI | Streamlit | 웹 대시보드 |
| 데이터 | pandas | 집계·필터·변환 |
| 차트 | Altair | 라인·막대·누적 막대·도넛 |
| 데이터 소스 | Google Sheets (CSV Export) | 실적·객실 |
| HTTP | requests + certifi | 시트 CSV 수집 (SSL) |
| AI (선택) | OpenAI API | 리뷰 요약 (`lib/ai_reviews.py`) |
| 개발 도구 | Cursor (Coding + AI Agent) | 구현·디버깅 |

`requirements.txt`: streamlit, pandas, altair, requests, certifi, openai

## 주요 설정 (`config.py`)

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `REVENUE_SHEET_ID` | (실적 시트 ID) | 실적 Google Sheet |
| `ROOMS_SHEET_ID` | (객실 시트 ID) | 객실 Google Sheet |
| `HOTEL_NAME` | ○○온천관광호텔 | 헤더 표시명 |
| `REFRESH_SECONDS` | 60 | `@st.cache_data` TTL |
| `RECENT_DAYS` | 7 | 실적관리 조회 일수 |

## 집계 로직 요약

| 화면 | 집계 단위 | 비고 |
|------|-----------|------|
| Daily Performance | 최신 월 일별 | 요약 탭 |
| 실적관리 | 최신일 포함 역순 7일 | 일별 원천 |
| 월별 실적 | `YYYY-MM` | 월 합계 |
| 일자별 매출 | 선택 월 1일~말일 | 실적 탭 |
| AX 연중 분석 | 역 12개월 | 인사이트 탭 |
| KPI 주간매출 | 완료 주 vs 전주 | 전주 대비 |

## 캐싱·성능

- `fetch_dashboard_data()`: `@st.cache_data(ttl=REFRESH_SECONDS)`
- 자동 `st.rerun()` 주기 새로고침 없음 (화면 깜빡임 방지)
- 데이터 갱신: 사용자 브라우저 새로고침
- Sheets fetch: `requests` + `trust_env=False` (로컬 프록시 우회)
