# 대시보드 화면 설명 · 발표 스크립트

## 탭 구성

| 탭 | 내용 |
|----|------|
| **요약** | KPI 4종 + Daily Performance (최신 월 일별 스냅샷) |
| **실적** | 실적관리(최근 7일) → 월별 실적 → 일자별 매출 |
| **객실운영** | 청소·검수 현황, 객실 리스트 |
| **VOC** | 평점·불만 카테고리 요약 |
| **AX·인사이트** | Executive 분석, 교차 권고, 리뷰 키워드·AI/규칙 요약 |

상단 탭 네비게이션은 PC·태블릿·모바일에서 가로 스크롤로 전환합니다.

---

## 1. 헤더

- **호텔명** (`config.py`의 `HOTEL_NAME`)
- **데이터 기준 시각** — 마지막 로드 시점

> "○○온천관광호텔 경영 현황을 한 화면에서 봅니다."

---

## 2. KPI 카드 (요약 탭, 4개)

| KPI | 설명 | 비교 |
|-----|------|------|
| 월매출 | 당월 합계 매출 | 전월 대비 % |
| 점유율 | 당월 평균 객실 점유율 | 전월 대비 % |
| ADR | 객실매출 ÷ 판매 객실수 | 전월 대비 % |
| 주간매출 | 기준 주(월~일) 합계 | 전주 대비 % |

- 단위: 매출·ADR는 **천원**, 점유율은 **%**
- 주간매출 카드 하단에 해당 주 라벨 표시 (예: `05/26(월)~06/01(일)`)
- 카테고리별 급락(전월 -10% 이하) 시 KPI 카드에 경고 배지 표시

> "경영 스냅샷은 월 KPI, 단기는 주간매출로 같이 봅니다."

---

## 3. Daily Performance (요약 탭)

- **오늘의 실적** — 데이터 최신 월 기준 일별 KPI 4종 (2×2 카드)
- **매출 구성 도넛** — 객실 / 온천 / F&B 비중
- 전일 대비 증감률 표시

> "요약 탭에서 당월 일별 스냅샷을 빠르게 확인합니다."

---

## 4. 실적관리 (실적 탭)

**부제:** 일별 원천 · 최근 7일 운영 실적

### 일별 매출 추이
- 최근 7일(데이터 최신일 포함) 합계 매출 라인 차트

### 일별 매출 구성
- 동일 7일, 일자별 **객실 / 온천 / F&B** 누적 막대

### 최근 7일 일별 상세
- 일자, 객실, 온천, F&B, 합계 표 (천원, 콤마)

> "일별 시트 데이터를 그대로 모아, 최근 일주일 추세와 구성을 봅니다."

---

## 5. 월별 실적 (실적 탭)

- **월별 매출 추이** + 월 선택 (selectbox 또는 표 행 클릭)
- **전월 대비 실적** — 당월 vs 전월, 항목별 증감 차트·표
- **월별 매출 구성비** — 선택 월 객실/온천/F&B

> "월별은 결산·경영 보고용입니다."

---

## 6. 일자별 매출 (실적 탭, 선택 월)

- 선택한 월의 **1일~말일** 일별 매출 라인 차트
- 동일 기간 일별 상세 테이블

> "월을 선택하면 해당 월의 일별 패턴을 드릴다운합니다."

---

## 7. 객실 청소 현황 (객실운영 탭)

- 불량률, 청소 미실시 건수
- 담당자별 총 건수·불량 건수 차트
- **객실 리스트**: 담당자 / **청소상태** / 검수결과 필터

데이터: 객실관리 Google Sheets (`Rooms` 탭)

---

## 8. 고객의소리 (VOC 탭)

- 평균 평점, 리뷰 수
- 카테고리별 불만 건수 차트
- 데이터: `data/voc.py` (프로토타입 샘플)
- 상세 리뷰 분석은 **AX·인사이트** 탭으로 연결

---

## 9. AX·인사이트 (인사이트 탭)

| 섹션 | 내용 |
|------|------|
| Executive Briefing | 경영 1줄 요약 + 핵심 bullet |
| 연중 매출 분석 | 역 12개월 계절·부문·분기 차트/표 |
| 마케팅 전략 | 사업부문·계절 기반 권고 카드 |
| 운영 전략 | 객실 품질·VOC 연계 권고 |
| AX 인사이트 | 실적·객실·VOC 교차 이상징후 카드 |
| 고객 리뷰 분석 | 평점 분포, 키워드, 규칙/AI 요약 |

- 분석 기간: 최신 데이터 월 포함 **역 12개월** (예: 2025.06~2026.05)
- 데이터: 실적·객실(Google Sheets), VOC(`data/voc.py`)
